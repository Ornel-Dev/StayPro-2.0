Vista General

Bienvenido al panel de control. Esta sección ofrece un resumen en tiempo real del estado del hotel. Utilice las tarjetas para ver los indicadores clave de rendimiento (KPIs) y el gráfico para entender la distribución actual de las habitaciones. Esta vista está diseñada para darle una comprensión inmediata de las operaciones diarias.

Gestión de Reservas

Aquí puede gestionar todas las reservas de los clientes. La tabla muestra las reservas confirmadas, en curso y canceladas. Utilice el botón "Nueva Reserva" para añadir una nueva entrada o las acciones en cada fila para realizar check-in o cancelar una reserva existente. Cada acción clave requerirá una confirmación para evitar errores.

Estado de Habitaciones

Esta vista proporciona un mapa visual del estado actual de todas las habitaciones del hotel. Cada tarjeta representa una habitación y su color indica su estado: disponible, ocupada, en limpieza o en mantenimiento. Puede realizar acciones rápidas como enviar una habitación a mantenimiento o marcarla como lista después de una limpieza.

Gestión de Limpieza

Diseñada para el personal de limpieza, esta sección muestra únicamente las habitaciones que requieren atención. Las tarjetas indican si una habitación está pendiente de limpiar o si el proceso ya ha comenzado. Utilice los botones para actualizar el estado de la limpieza en tiempo real, asegurando una comunicación eficiente con recepción.

Informes y Analíticas

Esta es el área de inteligencia de negocio del hotel. Los gráficos interactivos le permiten analizar tendencias clave a lo largo del tiempo. Explore los ingresos generados, la tasa de ocupación por tipo de habitación y los patrones de consumo para tomar decisiones estratégicas informadas y optimizar las operaciones del hotel.
